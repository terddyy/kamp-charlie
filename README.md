# Kamp Charlie-nae Web App

## Local development

Prerequisites: Node.js and npm.

Steps:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm i
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Deploy to your preferred provider (e.g., Vercel, Netlify, Cloudflare Pages). Build with:

```sh
npm run build
```
